# 靜態資料檔案目錄

此資料夾用於存放預處理好的單字資料檔案。

## 檔案格式

所有資料檔案使用 JSON 格式，包含以下結構：

```json
{
  "version": "2.0",
  "generated_date": "ISO日期格式",
  "total_words": 總單字數,
  "levels": {
    "300以下": { "start": 1, "end": 200 },
    "300-500": { "start": 201, "end": 700 },
    // ... 其他等級
  },
  "words": [
    {
      "word": "單字",
      "chinese": "中文解釋",
      "phonetic": "/音標/",
      "part_of_speech": "詞性",
      "level": "等級",
      "common_phrases": [
        {
          "phrase": "常用詞句",
          "chinese": "中文翻譯"
        }
      ],
      "toeic_example": {
        "sentence": "例句",
        "chinese": "例句翻譯"
      }
    }
  ]
}
```

## 使用方式

1. 使用 admin.html 上傳 TXT 檔案
2. 處理完成後點擊「產生靜態資料檔」
3. 將產生的 JSON 檔案放置在此資料夾中
4. 學習系統會自動讀取此資料夾中的檔案

## 等級分配

- **300以下**: 1-200 (最基礎單字)
- **300-500**: 201-700 (基礎商業單字)
- **500-600**: 701-1300 (中級商業單字)
- **600-700**: 1301-2000 (進階商業單字)
- **700-800**: 2001-2500 (專業商業單字)
- **800-900**: 2501-2800 (高級專業單字)
- **900以上**: 2801-3000 (最高級單字)
